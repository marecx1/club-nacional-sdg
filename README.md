# Club Manager Pro PY 🇵🇾

Sistema web de administración deportiva, social y financiera profesional para clubes y ligas en Paraguay.

## Características Clave
*   **Diseño Premium SaaS:** Interfaz oscura ultra-moderna, fluida y responsive.
*   **Gestión Integral de Socios:** Carnet digital, códigos QR, control de estados (Activo, Moroso, Suspendido).
*   **Finanzas en Tiempo Real:** Gestión de caja diaria, ingresos y egresos clasificados en Guaraníes (`Gs.`).
*   **Facturación PDF:** Descarga de recibos profesionales para cobro de cuotas y servicios.
*   **Automatización de Cobros:** Plantillas personalizadas para envío directo por WhatsApp a socios morosos.
*   **Dashboard Estadístico:** Gráficos de barra e ingresos con `Chart.js`.

---

## Cómo Ejecutar el Proyecto
Este proyecto es una aplicación web SPA estática de alto rendimiento.
1. Abre el archivo `index.html` en tu navegador.
2. Utiliza las siguientes credenciales para probar los diferentes roles:
   *   **Administrador:** Usuario: `admin` | Clave: `admin123`
   *   **Tesorero:** Usuario: `tesorero` | Clave: `teso123`
   *   **Secretario:** Usuario: `secretario` | Clave: `secre123`
   *   **Consulta:** Usuario: `consulta` | Clave: `con123`
